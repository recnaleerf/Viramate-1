Known Issues

  • Clicking skill buttons will fail to queue a skill if a skill animation completes playing while you are pressing the button. This is a bug in the game (not Viramate, for once!) and I cannot fix it.
  • The player's first skill (i.e. 1q) has no hotkey and the OK button in summon use popups has no hotkey. This is intentional for anti-cheat reasons.
  • The AP and BP meters in the popup are incorrect if you recently spent AP or BP
  • Popup item buttons get stuck if a network error occurs
  • Sidebar does not work if you are using a DMM account
  • Debuff timers flicker when updated
  • The HL raids menu is visible even if your rank is too low to start a HL raid
  • Please either play with 'Animation Settings' set to 'Lite', or with 'Resolution Settings' set to 'High'. Any other settings are unsupported.
  • Item watch buttons may not appear unless you have visited your Supplies page recently
